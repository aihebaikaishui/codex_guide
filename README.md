这是一个面向 OpenAI Codex 使用者的中文文档仓库，主要整理：

## 建议使用方式

1. 先读主指南，理解四件套的关系。
2. 按 `docs/project_brief.md` 先把项目目标写清楚。
3. 再用 `docs/project_status.md` 记录当前进展。
4. 每次让 Codex 做事时，尽量给出明确范围和验收标准。

- 尽量保持中英术语一致，便于检索和协作。
